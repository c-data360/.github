## Stack technique

### Web & Cloud

| Outil | Usage |
| --- | --- |
| Next.js · TypeScript | Applications web et sites institutionnels |
| Turborepo · pnpm | Monorepos multi-apps |
| Tailwind CSS · Shadcn/UI | Design system et interfaces |
| Vercel | Déploiement et hosting web |
| Supabase | Base de données, Auth, Storage (Postgres) |
| Prisma | ORM |
| Auth.js · Payload CMS | Authentification et gestion de contenu |
| Resend | Emails transactionnels |
| Mux | Upload et streaming vidéo |

### Mobile

| Outil | Usage |
| --- | --- |
| Flutter | Applications mobiles iOS & Android |
| React Native / Expo | Applications multiplateformes (migration à moyen terme) |
| Python / FastAPI | Backend mobile |
| Spring Boot | Backend mobile alternatif |
